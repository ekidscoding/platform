# Урок 0

## Ціль

Навчитись користуватись редактором. Перевірити налаштування середовища програмування

## Редактор коду

Редактор коду має візуальну та функціональну схожість на `VSCode`

Редактор підтримує функцію "**Історія**"

- `Ctrl + Z` скасує випадкову дію
- `Ctrl + Shift + Z` застосує дію повторно

Редактор підтримує **багато курсорів**:

1. Зажміть кнопку `Alt`
2. Помістіть курсор у відповідне місце та клікніть Лівою Кномкою Миші (**ЛКМ**) -- з'явиться додатковий курсор
3. Натисніть `Esc` щоб повернутись до єдиного курсору

Деякі корисні комбінації клавіш:

- `Ctrl + /` - закоментувати/розкоментувати
- `Ctrl + D` - виділити наступну копію слова
- `Alt + Down` / `Alt + Up` - перемістити рядок тексту Вниз або Вгору
- `Alt + Shift + Down` / `Alt + Shift + Up` - скопіювати рядок тексту Вниз або Вгору
